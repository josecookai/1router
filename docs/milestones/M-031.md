## Source

## Acceptance Criteria
1. 提供按 `provider + model` 分组的月账单 line items。
2. 每个 line item 返回 `quantity`, `unit_price`, `subtotal`, `platform_fee` 字段。
3. 账单 totals 与 usage 聚合可对账（line items 汇总等于 totals）。
4. 补充分组与总额对账测试，以及 endpoint 集成测试。
5. `make test`, `make coverage`, `make ci` 全绿且覆盖率达标。

## Scope
- In scope: in-memory 月账单聚合、line item schema、route + tests、OpenAPI/types。
- Out of scope: 税费、折扣券、账单状态机、持久化发票存储。
