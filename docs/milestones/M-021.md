# Milestone M-021

## Source
- Issue: #46
- Area: area/router

## Acceptance Criteria
1. `/v1/responses` 支持 `Idempotency-Key` 请求头；同 key + 同 payload 返回相同逻辑响应引用。
2. 同 key + 不同 payload 返回确定性的冲突错误（409）并使用共享 error envelope（含 `request_id`）。
3. 路由测试覆盖首次请求、重放请求、同 key 异 payload 冲突路径。
4. `make test`, `make coverage`, `make ci` 全部通过且覆盖率达标。

## Scope
- In scope: in-memory 幂等存储、payload 指纹校验、route 测试、OpenAPI 同步。
- Out of scope: 持久化幂等存储、跨进程一致性、TTL/清理策略。

## Test Plan
- Route: first request with idempotency key
- Route: replay with same key + same payload
- Route: same key + different payload -> 409 envelope
- CI: `make test`, `make coverage`, `make ci`
