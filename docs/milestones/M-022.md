# Milestone M-022

## Source
- Issue: #47
- Area: area/router

## Acceptance Criteria
1. 增加四个可选路由预设：`cost`、`latency`、`success`、`balanced`。
2. 评分决策逻辑可测试且可解释（返回 preset 与候选评分元数据）。
3. `/v1/responses` 在不破坏现有兼容性的前提下支持预设路由并返回选路信息。
4. 覆盖预设评分单测与路由级回归测试。
5. `make test`, `make coverage`, `make ci` 全绿，覆盖率达标。

## Scope
- In scope: weighted preset scoring、`/v1/responses` 元数据扩展、测试。
- Out of scope: 实时性能采样、动态权重学习、跨请求状态持久化。

## Test Plan
- Unit: 预设评分在固定输入下选择结果确定。
- Integration: `/v1/responses` 返回 preset 元数据并保留现有响应形状兼容。
- CI: `make test`, `make coverage`, `make ci`
