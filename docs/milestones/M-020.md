# M-020 Control-plane Contract Mocks Sync

## Status
- in_progress

## Scope
- Sync generated contract mocks with OpenAPI for:
  - `/api/models`
  - `/api/orgs/{orgId}/usage`
  - `/api/policies`
- Add tests to detect required-field drift in generated mocks.
- Keep generated OpenAPI artifacts committed and `openapi:check` clean.

## Acceptance Criteria
- Generated mocks include `/api/models`, `/api/orgs/{orgId}/usage`, `/api/policies`.
- Contract tests fail if required fields are missing from those mocks.
- Additive/backward-compatible fields are preserved.
- `npm run openapi:check`, `make test`, `make coverage`, and `make ci` pass.

## Validation Plan
- `npm run openapi:generate`
- `npm run openapi:check`
- `npm test -- test/control-plane-contract-mocks.test.ts`
- `make test`
