# Milestone M-025

## Source
- Issue: #51
- Area: area/router

## Acceptance Criteria
1. 路由候选支持区域约束（`US`/`EU`/`APAC`）过滤能力。
2. 带 region 偏好请求只选择合规候选；不合规候选在元数据里可解释地标注被排除原因。
3. 当指定 region 无合规候选时，执行可解释回退策略（fallback 到全量候选并标记回退）。
4. 测试覆盖：合规筛选、回退场景、无 region 偏好保持现有行为。
5. `make test`, `make coverage`, `make ci` 全绿且覆盖率达标。

## Scope
- In scope: region-aware 过滤 + fallback 元数据 + 测试。
- Out of scope: 真实 provider region 自动发现、地理合规策略中心。

## Test Plan
- Unit: region filtering and fallback decision behavior.
- Route: `/v1/responses` respect region preference and returns explainable metadata.
- CI: `make test`, `make coverage`, `make ci`.
