# Milestone M-030

## Source
- Issue: #61
- Area: area/router

## Acceptance Criteria
1. 触发阈值后 provider 自动进入 drain 状态（不可参与路由）。
2. 路由决策会排除被 drain 的 provider，并在响应元数据暴露排除列表。
3. cooldown 到期后 provider 自动恢复可选。
4. 测试覆盖 drain 触发、排除生效、cooldown 恢复。
5. `make test`, `make coverage`, `make ci` 全绿且覆盖率达标。

## Scope
- In scope: in-memory incident store、responses 路由过滤、测试、OpenAPI/types 同步。
- Out of scope: 持久化 incident 状态、跨实例共享和控制面配置化。
