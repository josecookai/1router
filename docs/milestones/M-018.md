# Milestone M-018

## Source
- Issue: #38
- Area: area/billing

## Acceptance Criteria
1. `/api/orgs/:orgId/usage` 对 `from/to/group_by` 边界做确定性校验并在非法输入时返回 400。
2. `from > to`、非法日期、非法 `group_by` 都返回共享 error envelope（含 `request_id`）。
3. 合法但空数据窗口返回稳定响应（`totals` 全 0，`buckets` 空数组），不崩溃。
4. 覆盖边界与回归场景测试并通过门禁。

## Scope
- In scope: usage query 边界校验、错误映射、一致性测试扩展。
- Out of scope: DB schema/聚合实现变更、结算业务逻辑调整。
