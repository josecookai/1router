## Scope
- Enforce role-based access checks for control-plane API endpoints.

## Acceptance Criteria

- 403 response includes role and scope context in `error.details`.
- Owner/admin/developer/billing/viewer matrix covered by tests.
- Minimal and backward-compatible implementation with centralized policy map.
- `make test`, `make coverage`, and `make ci` pass.
