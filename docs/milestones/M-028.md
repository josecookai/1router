# M-028 Minimal Orgs/Projects CRUD

## Status
- in_progress

## Scope
- Add minimal control-plane CRUD endpoints for organizations and projects:
  - `GET/POST /api/orgs`
  - `PATCH /api/orgs/:orgId`
  - `GET/POST /api/projects`
  - `PATCH /api/projects/:projectId`
- Reuse shared validation error envelope for malformed requests.
- Keep OpenAPI and generated contract types synchronized.

## Acceptance Criteria
- Minimal org/project create/list/update routes are implemented.
- Malformed payloads return shared envelope with `INVALID_REQUEST`.
- Route tests cover create/list/update and malformed payloads.
- OpenAPI schema includes org/project paths and component schemas.
- `make test`, `make coverage`, and `make ci` pass.

- `npm test -- test/org-projects-routes.test.ts`
