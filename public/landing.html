API Compatibility

Keep your SDK. Replace the base URL.

1router mirrors OpenAI-style request/response shapes and adds router metadata under a dedicated field. 

import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.ROUTER_API_KEY,
  baseURL: "https://api.1router.example.com/v1"
});

const res = await client.chat.completions.create({
  model: "gpt-4o-mini",
  messages: [{ role: "user", content: "Summarize in JSON" }],
  stream: false,
  router: { strategy: "balanced" }
});

console.log(res.choices[0].message.content);
console.log(res.router.provider);
